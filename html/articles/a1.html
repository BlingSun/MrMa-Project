2. 解决因子元素设置float: left导致父元素（如果未设置高度）高度为0

css代码为：

                .clr::before,
                .clr::after
                {
                    content: "";
                    display: block;
                    clear: both;
                    height: 0;
                    overflow: hidden;
                    visibility: hidden;
                }

给需要清除浮动的元素加上类.clr即可

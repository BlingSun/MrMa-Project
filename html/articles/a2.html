点击返回顶部（动画）


实现代码：

                $('.click_to_top_static').on('click', function() {
                    $(window).scrollTop(0);
                });

                $('.click_to_top_animate').on('click', function() {
                    $('html, body').animate({'scrollTop': '0'}, 500);
                });

注意：以上所有代码均在且只在ie9++上运行通过（其他现代浏览器自然可以运行）
